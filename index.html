World downloader

Clones a Minecraft multiplayer world from a remote server to your singleplayer folder

WorldDownloader 1.8c for Minecraft 1.8 using MCP 9.10

A minecraft mod that downloads a copy of a world on a multiplayer server for personal singleplayer use. This is a fork of @dslake's original version, updated to 1.8.

You can also try @uyjulian's liteloader version (which is based off of this one).
